## Sistema de Gestão de Biblioteca 

- Um sistema feito para praticar e Aperfeiçoar Java POO

# Funcionalidades
- Registro de Membro
- Registro de Livro
- Fazer Empréstimos De Livro
- Devoluções
- Histórico de Empréstimos(Devolvidos ou Não)
- Empréstimos Atrasados
- Empréstimos Feitos por um Membro
- Opções de Mostrar Livros e Membros Cadastrados


# Conceitos
- Programação Orientada a Objetos
- Encapsulamento
- Collections('ArrayList')
- Organização em Múltiplas Classes
- Relacionamento entre Objetos

# Estrutura
```
- Classes
  - Main
  - Library 
  - Book
  - Loan(Empréstimo)
  - Member
- Library
  - Representa a biblioteca
  - Gerencia Book, Member e Loan
- Book
  - Representa um livro
- Member
  - Representa um Membro
- Loan 
  - Representa um Empréstimo
``` 
## 🚀 Como executar o projeto

### 1. Pré-requisitos
- Java 21+ instalado
- IntelliJ IDEA (ou outra IDE Java)

### 2. Executando no IntelliJ
1. Abra o projeto no IntelliJ IDEA
2. Aguarde o Maven/Gradle (se tiver) carregar o projeto
3. Vá até a classe `Main`
4. Clique em **Run (▶)** ou execute o método `main`

### 3. Usando o sistema
- O programa roda no terminal
- Siga o menu interativo para usar o sistema de biblioteca
